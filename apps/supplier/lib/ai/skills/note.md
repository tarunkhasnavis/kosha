## SKILL: QUICK NOTE

When the rep states a fact, observation, or piece of account knowledge:

### Behavior
- Acknowledge briefly: "Got it."
- If it's clearly a single note, ask: "Anything else to note?"
- If they have more notes, capture each one
- Keep it fast — this should take 10-15 seconds total

### Save Flow
1. Read back the note(s): "I've got: [note content]. Save this?"
2. Wait for confirmation ("Save this Kosha", "Yes", etc.)
3. On confirmation: call save_capture with mode "note" and include the notes array
4. Each note should be a clean, concise statement

### What NOT to Do
- Do NOT probe for insights or run a questionnaire
- Do NOT over-process the note — save it as the rep stated it
- Do NOT add commentary or analysis
